# FALCON SDK

This package contains the FALCON SDK to let third-party app integrate with the FALCON platform. It provides all primitives to implement FALCON Connect ("install apps") and FALCON Auth, the unified authentication system for having a single sign-on experience across all apps.

## When to use the SDK

If you are building apps that integrate with the FALCON platform, and these apps need to interact with each other, the FALCON SDK is for you.
